# Korean Skincare Website

## Description
A modern and responsive website showcasing popular Korean skincare products. Users can browse products, learn about different skincare categories, and explore routines for various skin types.

## Features
- Browse Korean skincare products
- Search and filter products
- View detailed product information
- Responsive design for desktop and mobile
- Easy-to-use navigation

## Technologies Used
- HTML5
- CSS3
- Php

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your web browser.
